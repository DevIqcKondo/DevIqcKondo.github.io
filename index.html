Sobre mim

  • Leitura

  • Escrita

    Escrever sempre foi algo fácil para mim. Sempre amei as palavras e como elas me traduziam.

  • Fotografia

    Sempre gostei de tirar foto de coisas, lugares, flores. A fotografia é uma maneira de sentir, de tocar, de amar. O que capturamos em uma imagem é capturado para sempre.
